Right-angled trigonometry

How tall is the building?

Using trigonometric ratios to find missing heights.

Start the lesson

Lesson question

Can we calculate the height of a building without climbing it?

Label sides → Choose ratio → Calculate height → Explain answer

Learning objective

Pupils will learn to choose and apply sine, cosine or tangent to find a missing side in a right-angled triangle.

Prior knowledge

  • Right-angled triangles
  • Hypotenuse, opposite and adjacent
  • Basic ratio and calculator use
  • Rearranging simple formulae

Interactive example

For a height problem, tangent links the opposite side to the adjacent side.

tan(θ) = opposite ÷ adjacent
h = 20 x tan(35 degrees) = 14.00 m

Teacher modelling script

"We know the distance from the building is the adjacent side. The unknown height is the opposite side. Because tangent connects opposite and adjacent, I will use tan."

tan(35 degrees) = h / 20
h = 20 x tan(35 degrees)
h = 14.0 m
